HTML

HTML (Hypertext Markup Language) jest to kod używany do tworzenia struktury strony i jej zawartości.

Nie jest językiem programowania jest językiem znaczników (ang. markup language). HTML składa się z serii znaczników (tagów), których używa się do zamknięcia, opakowania różnych części treści, tak aby wyglądały i/lub działały w określony sposób. Z pomocą tagów możesz ze słów czy obrazów zrobić linki do innych stron, sprawić by były napisane kursywą, większą czcionką, wytłuścić je itd.

div

jest rodzajem pojemnika na treść. Nie ma on żadnego wpływu na treść ani układ graficzny, dopóki nie zostanie ostylizowany z użyciem CSS. Element <div> jako "czysty" pojemnik niczego nie reprezentuje. Zamiast tego, jest on używany do grupowania zawartości.

CSS

CSS jest językiem określającym, jak dokumenty HTML mają być prezentowane użytkownikowi.

Nazwa „kaskadowe arkusze stylów” wynika z faktu, iż gdy reguły CSS wykluczają się wzajemnie w arkuszu zewnętrznym, arkuszu wewnętrznym oraz na poziomie elementów HTML, priorytet stylów ustalany jest hierarchicznie.

Jak podłączyć CSS do HTML

 <head>
  ...
  <link rel="stylesheet" href="styles.css">
  ...
</head>

Selektory

  • tag- div, p, input, button
  • klasa - .box
  • id - #title, #container 
<div class="box" id="container"></div>
.box {
  background-color: red;    
  font-size: 12px;    
  color: white; 
}

Margines i padding 

 .content {
  margin: 5%;  /* wszystkie strony margines 5% */
  padding: 5%;  /* wszystkie strony padding 5% */
}

.content {
  margin: 10px 20px;            /*  górny i dolny margines 10px, lewy i prawy margines 20px */
  padding: 10px 20px;            /*  górny i dolny padding 10px, lewy i prawy padding 20px */
}                                

.content {                       /*  górny margines i padding 10px */
  margin: 10px 3% 20px;         /*  lewy i prawy margines i padding 3% */
  padding: 10px 3% 20px;        /*  dolny margines i padding 20px */
}                                

.content {                       /*  górny margines i padding 10px */
  margin: 10px 3px 30px 5px;    /*  prawy margines i padding 3px */
  padding: 10px 3px 30px 5px;    /*  lewy margines i padding 5px */
}                                /*  dolny margines i padding 30px */

Dziedziczenie

A E { ... }
Dowolny element E, który jest potomkiem elementu A (to znaczy jest dzieckiem albo dzieckiem dziecka, itp.))
A > E { ... }
Dowolny element E, który jest dzieckiem elementu A.
A + E { ... }
Dowolny element E, który jest następnym z rodzeństwa elementu B (to znaczy: następnym dzieckiem tego samego rodzica).

Pseudo klasy

Pseudo klasy pozwalają obsługiwać dodatowe efekty.
 selector:pseudo-klasa { ... }
:focus
Pozwala stylować aktualnie aktywny element.
:hover
W ten sposób stylujemy element na który najechało się kursorem.
:nth-child(n)
stylujemy n-ty tag.
:first-child
sprawia iż własności stosowane są do elementu tylko wtedy gdy jest on pierwszym węzłem potomnym swojego rodzica.

Flexbox

Flexbox to jednowymiarowa metoda rozmieszczania elementów w wierszach lub kolumnach, przedmioty rozciągają się aby wypełnić dodatkową przestrzeń lub kurczą się aby dopasować się do mniejszych przestrzeni.

Grid

CSS Grid Layout jest najlepszą metodą aby podzielić strone na części lub definiowanie relacji takich jak rozmiar, pozycja i warstwa, pomiędzy podstawowymi elementami HTML. Podobnie do tabel (ang. "tables"), rozmieszczenie grid (ang. "siatka") pozwala autorowi wyrównać elementy w kolumny i wiersze. Jednakowoż grid pozwala na wiele więcej ułożeń niż tabele.
